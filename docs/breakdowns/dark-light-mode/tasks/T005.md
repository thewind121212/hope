# T005: Build ThemeToggle Dropdown Component

## State: `pending`

## Action
Create `components/theme/ThemeToggle.tsx` — dropdown menu with Light/Dark/System options.

## Business Summary
The user-facing control that lets users pick their preferred theme. Clear visual feedback on current selection.

## Logic
- Trigger button: icon (sun/moon/monitor) reflecting current mode
- Dropdown menu with three options: Light, Dark, System
- Each option shows an icon + label
- Selected option is visually highlighted
- Clicking an option calls `setMode()` and closes menu
- Keyboard accessible (Enter/Space to open, arrow keys to navigate)

## Technical Logic
- Use existing `DropdownMenu` from `components/ui/DropdownMenu.tsx` if suitable, or build minimal custom dropdown
- Icons: use inline SVG or a small icon set (Lucide-style)
- Client component (`"use client"`)

## Testing
Manual: Click toggle → select each option → verify theme changes and selection state persists.
