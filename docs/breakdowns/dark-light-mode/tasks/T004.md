# T004: Create ThemeContext + useTheme Hook

## State: `pending`

## Action
Create `hooks/useTheme.ts` with React Context for theme state.

## Business Summary
Enables any component to read the current theme and change it. The toggle and any future theme-aware components depend on this.

## Logic
- `ThemeProvider` wraps the app, holds `mode` state (`'light' | 'dark' | 'system'`)
- On mount, read from `getStoredTheme()` (default: `'system'`)
- `setMode(mode)` updates state, calls `setStoredTheme()`, and applies class to `<html>`
- `useTheme()` returns `{ mode, resolvedTheme, setMode }`
- `resolvedTheme` is always `'light' | 'dark'` (resolved from mode)

## Technical Logic
```ts
// On mode change:
const resolved = resolveTheme(mode);
document.documentElement.classList.toggle('dark', resolved === 'dark');
setStoredTheme(mode);
```
