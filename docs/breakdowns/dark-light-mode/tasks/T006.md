# T006: Integrate ThemeToggle into Header Layout

## State: `pending`

## Action
Modify `app/layout.tsx` to include `ThemeScript`, `ThemeProvider`, and `ThemeToggle`.

## Business Summary
Makes the theme toggle accessible to users from the main header. Completes the integration of all theme components.

## Logic
1. Import `ThemeScript` and render in `<head>` or at start of `<body>`
2. Wrap content with `ThemeProvider`
3. Add `ThemeToggle` to header, positioned far-right
4. Ensure header layout uses flexbox with `justify-between` or similar

## Technical Logic
```tsx
<html lang="en">
  <head>
    <ThemeScript />
  </head>
  <body>
    <ThemeProvider>
      <header>
        <div className="flex items-center justify-between">
          <h1>Bookmark Vault</h1>
          <ThemeToggle />
        </div>
      </header>
      ...
    </ThemeProvider>
  </body>
</html>
```

## Testing
Manual: Load app → toggle visible in header → clicking works → refresh preserves choice.
